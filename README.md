## Ссылка на опубликованный проект
https://dmitryfatkhullin.github.io/shfe-diplom/

Frontend
HTML5 — семантическая разметка страниц
CSS3 — стилизация, адаптивная верстка
JavaScript (ES6+) — логика приложения, асинхронные запросы, работа с DOM
Bootstrap 5 — адаптивная сетка и базовые компоненты интерфейса

Библиотеки
qrcode-generator — генерация QR-кодов для электронных билетов
Fetch API — взаимодействие с backend-сервисом
SessionStorage — временное хранение данных между страницами

Инструменты разработки
Git — контроль версий
GitHub — хостинг репозитория и GitHub Pages
ESLint — проверка качества кода
Chrome DevTools — отладка и тестирование адаптивности
Visual Studio Code — среда разработки

Backend
REST API — https://shfe-diplom.neto-server.ru/
Формат данных: JSON
Методы: GET, POST, DELETE
